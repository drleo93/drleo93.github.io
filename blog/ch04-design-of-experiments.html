Ch 4: Design of Experiments & Sensitivity Analysis

Chapter 4 of 23  ·  ~30 min read  ·  Invest as much here as in the model

TL;DR

  • The design of experiments determines everything downstream. A poor experimental design cannot be compensated by a sophisticated surrogate.
  • For surrogate construction: use Sobol' or optimised LHS with \(n \geq 10d\) for GP, \(n \geq 50d\) for XGBoost. Never use full factorial above \(d = 4\).
  • Run global sensitivity analysis on a cheap proxy before collecting expensive training data. Identifying two inert inputs before a 200-point campaign saves €20,000–€100,000.
  • Sobol' total-effect index \(S_{T_i} < 0.01\): input is inert — fix it at nominal and reduce problem dimensionality.
  • D-optimal and G-efficiency metrics apply only to RSM (linear models). They are meaningless for GP and XGBoost — use space-filling metrics instead.
  • Active learning (GP max-variance acquisition) always improves any static initial design when additional evaluation budget is available.

1. The Information-per-Euro Principle

Every engineering experiment represents a finite budget. A test-bench run at a chemical pilot plant costs €500–€2,000 and takes 4–12 hours. A RANS CFD simulation costs 24–96 hours on a high-performance cluster. The DOE determines how much information is extracted per unit of this budget.

Three fundamental design properties
A design \(\mathcal{X} = \{\mathbf{x}_1,\ldots,\mathbf{x}_n\} \subset \mathbb{R}^d\) for surrogate modelling should satisfy:
  1. Space-filling: points cover the input domain uniformly.
  2. Non-collapsing: the projection onto any \(k\)-dimensional subspace is also space-filling, for \(k = 1,\ldots,d-1\).
  3. Orthogonality: columns of the design matrix are uncorrelated; enables independent estimation of main effects.
Classical factorial designs satisfy (3) but not (1) or (2). Space-filling designs satisfy (1) and (2) but not always (3). The correct type depends on the intended use.

2. Design Quality Metrics

Space-Filling Metrics

Maximin distance — larger is better; ensures no two points are unnecessarily clustered:

$$\Phi_{mm}(\mathcal{X}) = \min_{i \neq j} \|\mathbf{x}_i - \mathbf{x}_j\|_2$$

Average minimum distance (AMD) — less dominated by outlier pairs than maximin:

$$\Phi_{AMD}(\mathcal{X}) = \frac{1}{n}\sum_{i=1}^n \min_{j \neq i}\|\mathbf{x}_i - \mathbf{x}_j\|_2$$

Star discrepancy \(D^*\) — smaller is better; the Koksma-Hlawka inequality bounds integration error directly:

$$\left|\frac{1}{n}\sum_{i=1}^n f(\mathbf{x}_i) - \int f\,d\mathbf{x}\right| \leq D^*(\mathcal{X})\cdot V(f)$$

Low-discrepancy sequences achieve \(D^* = \mathcal{O}((\log n)^d/n)\) vs. \(\mathcal{O}(n^{-1/2})\) for random Monte Carlo.

Centred \(L_2\) discrepancy (\(CD^2\)): the recommended default for LHS optimisation — fast to compute, closed-form, rotation-invariant.

Model-Based (Optimality) Metrics — RSM Only

D-efficiency: minimises the volume of the confidence ellipsoid on \(\hat{\boldsymbol{\beta}}\). 100% = globally optimal. Use only for RSM — meaningless for GP/XGBoost.

Condition number \(\kappa(\mathbf{X}_p^\top\mathbf{X}_p)\): target < 100 for RSM. Above 1,000: numerical instability in OLS.

3. Classical Designs

Full Factorial and Fractional Factorial

A \(2^k\) full factorial evaluates all \(2^k\) combinations. Use when: \(k \leq 4\); goal is main-effect and two-way interaction estimation; budget allows all \(2^k\) runs.

A \(2^{k-p}\) fractional factorial uses \(2^{k-p}\) runs with resolution \(R\):

  • Resolution III: main effects confounded with 2FI — screening only, \(k \geq 7\)
  • Resolution IV: main effects clear; 2FI confounded with 2FI
  • Resolution V: main effects and 2FI clear — preferred for RSM

Central Composite Design (CCD) and Box-Behnken (BBD)

CCD augments a factorial with \(\pm\alpha\) axial (star) points and \(n_c\) centre runs. Total runs: \(2^k + 2k + n_c\). Rotatable with \(\alpha = (2^{k-p})^{1/4}\); face-centred (\(\alpha = 1\)) when corners are infeasible.

BBD places points at edge midpoints only — no vertex runs. Safe when all-extreme-level combinations are operationally infeasible (safety, cost, or feasibility constraints). For \(k = 3\): \(n = 12 + n_c\) runs.

4. Space-Filling Designs

Latin Hypercube Sampling (LHS)

An LHS partitions each of the \(d\) dimensions into \(n\) equal strata and places exactly one point per stratum per dimension, guaranteeing uniform marginal distributions.

  • \(CD^2\)-optimal LHS: minimises centred \(L_2\) discrepancy; fast column-pairwise swap algorithm — recommended default
  • Maximin LHS: maximises minimum pairwise distance; better space-filling; slower \(\mathcal{O}(n^2)\) per iteration
  • Orthogonal LHS: all 2D projections are also LHS; best balance; exists for \(n = p^2\) (prime power)

Sobol' Quasi-Random Sequences

Deterministic, low-discrepancy:

  • Discrepancy: \(D^* = \mathcal{O}((\log n)^d / n)\) vs. \(\mathcal{O}(n^{-1/2})\) for Monte Carlo
  • Reproducible (seed-based); best at \(n = 2^m\)
  • Scrambling (Owen scrambling) removes leftmost corner bias and enables unbiased variance estimation
  • Works well up to \(d \approx 40\); use Faure or Niederreiter for \(d > 40\)
from scipy.stats.qmc import Sobol
sampler = Sobol(d=5, scramble=True, seed=42)
X = sampler.random_base2(m=7)  # n=128 points in 5D

5. Global Sensitivity Analysis

Morris Screening (Elementary Effects)

Cheapest GSA approach — designed for screening \(d > 20\) inputs with only \(r(d+1)\) model evaluations:

$$EE_i(\mathbf{x}) = \frac{f(\mathbf{x} + \Delta\mathbf{e}_i) - f(\mathbf{x})}{\Delta}$$

Compute \(r \geq 10\) elementary effects per input from different starting points. Summary statistics: \(\mu_i^* = \mathbb{E}[|EE_i|]\) (importance) and \(\sigma_i = \mathrm{Std}(EE_i)\) (nonlinearity/interaction indicator).

Decision rule: if \(\mu_i^* < 0.1\max_j\mu_j^*\): candidate for elimination. If \(\sigma_i > \mu_i^*\): strong nonlinearity or interaction.

Sobol' Variance-Based Indices

$$S_i = \frac{\mathrm{Var}(\mathbb{E}[Y|X_i])}{\mathrm{Var}(Y)} \quad \text{(first-order)}, \qquad S_{T_i} = 1 - \frac{\mathrm{Var}(\mathbb{E}[Y|\mathbf{X}_{\sim i}])}{\mathrm{Var}(Y)} \quad \text{(total)}$$

Interpretation: \(S_{T_i} < 0.01\) — input is inert, fix at nominal. \(S_{T_i} \gg S_i\) — strong interactions. \(\sum_i S_i < 1\) — significant interactions present.

Example — Catalytic steam methane reformer (SMR)
8 inputs: \(T\), \(p\), \(S/C\), GHSV, \(\chi_{CH_4}\), \(\chi_{CO_2}\), \(\chi_{N_2}\), \(A_{mod}\).

Sobol' results: \(S_{T,T} = 0.58\) (temperature dominates — confirming Arrhenius rate control). \(S_{T,S/C} = 0.32\), \(S_{T,p} = 0.19\) through equilibrium effects. Feed composition variables (\(\chi_{CH_4}\), \(\chi_{CO_2}\), \(\chi_{N_2}\)) and \(A_{mod}\) all have \(S_{T_i} < 0.01\).

Action: fix 4 inert inputs at nominal, reduce from 8D to 4D, save 50% of experimental budget.

Estimation cost: Saltelli estimator requires \(n(2d+2)\) model evaluations for all first-order and total indices simultaneously. For expensive models: train a GP surrogate on \(n_{training} = 50d\) points, then sample \(10^6\) from the surrogate (cost \(\approx n_{training}\)).

Derivative-Based Global Sensitivity (DGSM)

For models with available gradients (surrogate or AD-enabled simulation):

$$\nu_i = \int_\mathcal{X}\!\!\left(\frac{\partial f}{\partial x_i}\right)^{\!2}\!d\mathbf{x} \approx \frac{1}{n}\sum_{j=1}^n\!\left(\frac{\partial f}{\partial x_i}(\mathbf{x}_j)\right)^{\!2}$$

DGSM bounds \(S_{T_i}\) from above. Requires \(n(d+1)\) evaluations with forward-mode AD — far cheaper than Saltelli for \(d > 20\).

Rule — Design selection by goal
  • RSM (polynomial model for ANOVA): CCD for \(d \leq 6\); fractional factorial for screening; BBD when corner runs are infeasible
  • GP surrogate: Sobol' + LHS with \(n \geq 10d\). Always scramble Sobol' sequences.
  • XGBoost / RF surrogate: Sobol' or LHS with \(n \geq 50d\)
  • Mixture constraints: simplex-centroid or Dirichlet LHS
  • High-dimensional screening (\(d > 10\)): Morris first, then Sobol' on reduced space
  • Adaptive / expensive model: active learning with GP max-variance acquisition

7. Bayesian Optimal Experimental Design (BOED)

The theoretical foundation: the Expected Information Gain (EIG) for a proposed experiment at \(\mathbf{x}^*\):

$$\text{EIG}(\mathbf{x}^*) = \mathbb{E}_{p(y^*|\mathbf{x}^*)}\!\left[D_{KL}\!\left(p(\boldsymbol{\theta} \mid y^*, \mathbf{x}^*) \;\big\|\; p(\boldsymbol{\theta})\right)\right]$$

For a Gaussian surrogate, EIG reduces to:

$$\text{EIG}(\mathbf{x}^*) = \tfrac{1}{2}\log\!\left(1 + \frac{k(\mathbf{x}^*,\mathbf{x}^*)}{\sigma_n^2}\right)$$

This is the GP max-variance acquisition function: run the experiment where the model is most uncertain. The Bayesian optimisation Expected Improvement (EI) is a special case where the parameter of interest is the global optimum location rather than the full parameter vector (Chapter 15).

Pitfall — Calling the statistician after the experiment
Fisher's 1938 warning remains accurate: a poorly designed experiment cannot be rescued by a sophisticated analysis. If you collect 200 points on a regular grid in a 5D space, the between-grid-line regions will be empty, and the GP will have high posterior variance everywhere it matters most (between training points). The statistical analysis will correctly report high uncertainty — but the experiment cannot be rerun. The DOE must be planned before any data is collected.

Chapter Summary

Key takeaways
  • Run sensitivity analysis before the main campaign: Morris screening costs \(r(d+1)\) evaluations. If it eliminates 3 inputs from 8, you reduced from \(10 \times 8 = 80\) to \(10 \times 5 = 50\) points for the GP — saving 37% of the budget.
  • Sobol' sequences are the default for GP and ML surrogates. They achieve lower discrepancy than random LHS and scale to \(d \approx 40\).
  • Full factorial above \(d = 4\) is never appropriate for surrogate construction — \(2^5 = 32\) points in 5D provides poor space-filling coverage compared to a 32-point Sobol' design.
  • D-efficiency is meaningless for GP/XGBoost. It assumes a linear model. Use \(CD^2\) or maximin for space-filling designs.
  • Active learning with GP max-variance acquisition: always add budget to sequentially refine the surrogate in regions of high uncertainty before making final design decisions.
  • Sobol' \(S_{T_i} < 0.01\): hard threshold for input elimination. Confirm with domain knowledge before fixing.
